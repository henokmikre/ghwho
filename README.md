# ghwho

GitHub-based user information lookup.

## Installation

### Prerequisites

```
pip install pycurl
```

```
curl -L http://github.com/henokmikre/ghwho/raw/master/ghwho.py > ghwho
```

Make it executable:

```
chmod +x ghwho
```

Move it to a bin:

```
mv ghwho /usr/local/share/ghwho
```

## Usage

```
ghwho henokmikre
```
